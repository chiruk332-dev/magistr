# Інформаційна система для вивчення тактичних знаків

Система реалізує каталог умовних тактичних знаків, навчальні картки, тестування, простий редактор оперативної схеми та адміністративний модуль.

## Технології
- Backend: Flask, Flask-SQLAlchemy, SQLite
- Frontend: React + Vite
- Графіка знаків: SVG

## Ролі
- `student` — перегляд довідника, навчання, проходження тестів, створення схем.
- `admin` — керування довідником знаків.

## Тестові облікові записи
- admin / admin123
- student / student123
