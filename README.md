# Інформаційна система для вивчення тактичних знаків

Навчальна програмна реалізація для магістерської роботи: **«Інформаційна система для вивчення тактичних знаків для оформлення оперативних документів»**.

## Що реалізовано

- Backend: Flask + SQLite.
- Frontend: React + Vite.
- Повний навчальний каталог умовних знаків, сформований за прикріпленим документом: 839 записів.
- Пошук і фільтрація знаків за категоріями.
- Навчальні картки.
- Тестування.
- Редактор оперативної схеми:
  - бібліотека знаків із пошуком;
  - додавання знаків на схему;
  - перетягування знаків мишею;
  - видалення знака зі схеми;
  - генерація умовної топографічної карти з населеними пунктами, дорогами, річкою, лісами та висотами;
  - збереження схеми у базу даних.
- Функціонал ручного додавання нових знаків прибрано.

> Примітка: графічні SVG-піктограми є навчальними стилізованими позначеннями, автоматично згенерованими для кожного запису каталогу. Назви та категорії винесено в базу для відпрацювання пошуку, навчання й редактора схем.

## Запуск backend

```bash
cd backend
python -m venv venv
venv\Scripts\activate      # Windows
# source venv/bin/activate  # Linux/macOS
pip install -r requirements.txt
python run.py
```

Backend буде доступний за адресою: `http://localhost:5000`.

## Запуск frontend

```bash
cd frontend
npm install
npm run dev
```

Frontend буде доступний за адресою, яку покаже Vite, зазвичай `http://localhost:5173`.

## Облікові записи

- admin / admin123
- student / student123

Авторизація в MVP не є обов’язковою для перегляду довідника та роботи з редактором.

## Оновлення зі старої версії

Якщо у вас уже запускалася попередня версія з 5 демо-знаками, достатньо запустити backend. Під час старту база автоматично синхронізується з повним каталогом.

Якщо frontend має старі залежності:

```powershell
cd frontend
Remove-Item -Recurse -Force node_modules
Remove-Item -Force package-lock.json
npm install
npm run dev
```
